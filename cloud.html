Cloud PLC 4.0

The Programmable Cloud PLC® offers unmatched adaptability, reliable connectivity and customizable solutions for your industry. Cloud PLC® can be utilized as an edge gateway, automation tool, data logger and edge processing node.

Supports industrial protocols like Modbus, MQTT, JSON, RESTful, TCP/UDP, SNMP and OPC, making integration easier than ever for IT and Automation engineers.

Features
  • 8× Isolated Analog Inputs (16 Bit, 860 SPS)
  • 2× Analog Output (0–10V), 2× 0–20mA Transmitter
  • 4× Isolated Digital Inputs & Outputs (24V)
  • 2× Relay (NO & C)
  • RS485, RS232 & USB Interface
  • WiFi / Bluetooth / BLE
  • JSON / MQTT / RESTful / OPC
  • SSL Secure Communication
  • 16GB Inbuilt Storage*
  • Power Supply: DC 9–36V
Open Source Development Environment

Cloud PLC® 4.0 enables building custom industrial solutions for monitoring and controlling PLC, SCADA, HMI, VFDs, motors, valves, energy meters, actuators, relays, encoders, RFID and fingerprint readers and industrial sensors.

Applications
  • Production & Process Monitoring
  • Utilities & Condition Monitoring
  • Environment Monitoring
  • Industrial Smart Grid
  • Leakage Detection
  • Generator & Cold Storage Monitoring
  • Calamity Warning Systems
  • Standard SCADA Applications
  • District Metering
  • Water Treatment & Green House
