R-LiFi

R-LiFi Nano v2 is an easy-to-use, plug-and-play evaluation platform for visible light communication applications in consumer, industrial, medical and IoT markets.

LiFi Nano consists of a transmitter module connected to LED lighting and a receiver module connected to the host system.

Features
  • Supports baud rates 38400 – 115200
  • Serial communication support
  • Distance support: 6 – 15 feet
  • Plug-and-Play configuration
  • Ceiling / wall mounted LED communication
