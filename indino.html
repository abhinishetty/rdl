THE NEXT GENERATION open source controls!

Indino 4.0 series defines a new way of transforming factories into smart / intelligent factories for efficient and easy remote monitoring operational status such as on/off, pressure and temperature.

Indino 4.0 supports industrial protocols like Modbus, MQTT, JSON, RESTful, TCP/UDP, SNMP, enabling seamless monitoring and integration through open source APIs.

Features
Processor:
  • Tensilica Xtensa 32-bit LX6 microprocessor
  • Clock frequency: up to 240 MHz, up to 600 DMIPS
  • ROM: 448 KB   SRAM: 520 KB
Co controller:
  • Atmega 2560, 16MHz, FLASH 256KB / STM32*
Digital IO:
  • 24V 8× Isolated enabled digital input
  • 24V 1 Amp 4× Isolated digital output / PWM
  • AC Isolation: 3750 VRMS
  • Contacts supported: DRY / WET
Analog IO:
  • 0–10V / 4–20mA 4× analog input
  • 2/3/4 terminal RTD – 1×
  • 2× K-Type thermocouple
Wired Connectivity:
  • RS485 MODBUS, RS232 & USB
  • RDL expansion bus
  • Memory: FRAM 25KB, SD Card up to 32GB
  • RTC: Built-in RTC for time-stamped logging
Wireless Connectivity:
  • Wi-Fi 802.11 b/g/n/e/i (up to 150 Mbps)
  • Bluetooth v4.2 BLE
  • GPRS / XBee / LoRa
Protocol:
  • TCP/IP, UDP, SNMP, MODBUS, FTP
  • RESTful, JSON & MQTT
Security:
  • WPA / WPA2, WFA & WAPI
  • Secure boot & flash encryption
  • Cryptography: AES, SHA-2, RSA, ECC, RNG
Power Supply:
  • DC 9–30V
Enclosure:
  • IP20 • Wall / DIN Rail mounting
  • Dimension: 108 × 41.2 × 20 mm
Indino 4.0 Products

Indino Digital IO

Indino main block which supporting 8x 24v isolated Digital INPUT and 4x 24V digital OUTPUT / PWM enabled with WIFI / BLE, RS232, RS485 communication and RTC, SDCARD for real time Data stamping.

Indino Analog IO

Add On Modular blocks for the Indino, for expanding the Analog IO (0-10V / 4-20ma), 1x RTD, 2x Thermocouple and supporting RS232, RS485 and 4x DO / PWM.

Indino Ethernet IO

Add On Modular blocks for the Indino, for expanding the Ethernet IO and supporting 6x 24v isolated Digital INPUT and 5x 24V digital OUTPUT.

Indino GPRS IO

Add On or independent blocks, for expanding the GPRS IO and supporting 8x 24v isolated Digital INPUT and 4x 24V digital OUTPUT / PWM enabled with RS232, RS485 communication and RTC, SDCARD for real time Data stamping.

Indino Xbee IO

Add On or independent blocks, for expanding the XBee IO and supporting 8x 24v isolated Digital INPUT and 4x 24V digital OUTPUT / PWM enabled with RS232, RS485 communication and RTC, SDCARD for real time Data stamping.

Indino LoRA IO

Add On or independent blocks, for expanding the LoRA IO and supporting 8x 24v isolated Digital INPUT and 4x 24V digital OUTPUT / PWM enabled with RS232, RS485 communication and RTC, SDCARD for real time Data stamping.

Open Source Programming IDE

ARDUINO IDE

The open-source Arduino Software (IDE) makes it easy to write code and upload it to the board. It runs on Windows, Mac OS X, and Linux. (www.arduino.cc)

ATMEL Studio

Integrated development platform (IDP) for developing and debugging all AVR® and SAM microcontroller applications using C/C++ or Assembly.

Arduino Compatible Compiler for LabVIEW

Compiler that takes LabVIEW programs and deploys them to Arduino™ compatible targets. (www.tsxperts.com)

ThingSpeak for IoT

ThingSpeak™ is an IoT analytics platform from MathWorks® for aggregating, visualizing, and analyzing live IoT data. (thingspeak.com)

RDL Plug and Play Configuration Manager

Every datalogger is shipped with Plug & Play Configuration Manager, enabling fast configuration with minimal technical expertise.

Soapbox Snap

Open-source PC-based automation platform with ladder logic editor and runtime support. (soapboxautomation.com)
